# SNAP Skills Repo

SNAP = SNAP's Not A Prompt. Agent Skills following the [agentskills.io](https://agentskills.io) spec.

Each top-level directory is a skill. Read a skill's SKILL.md before modifying it.

## Writing Style

Dense technical and philosophical wording — fewer tokens, higher impact. Sacrifice grammar for precision. Use heavy technical terms so the agent grasps intent without verbose explanation.

## Skill Modification

Always use `/skill-creator` when planning, creating, editing, or improving any skill.
